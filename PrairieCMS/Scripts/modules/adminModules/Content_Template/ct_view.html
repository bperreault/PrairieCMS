Pages
Select an existing page or create a new one.

The content template is the body of your web page.
The wrapper for your page commonly contains header logo and footer details, it will default to 'Mobile Web Wrapper' but you can select a different one on the WRAPPER tab.

  • Enter a name for this Page
    the page name is also the url path for this page ( home , contact , about )
  • Select the Wrapper for your page.
  • Enter a title for this Page
    the title is displayed in the browser top bar and should be descriptive of your page's functionality. (do you also want to add your web site titel following a dash?)
  • Select the role of people who can edit this page
    super admin's always have access
Enter the html for your page.
This html will be inserted into the chosen wrapper at the location specified by {content_template}
