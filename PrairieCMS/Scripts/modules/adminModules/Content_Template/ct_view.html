Select an existing content template or create a new one.

The content template is the body of your email.
The wrapper for your email content commonly contains header logo and footer details, it will default to 'Mobile Web Wrapper' but you can select a different one on the WRAPPER tab.

  • Select from existing Content Templates:    
  • or
  • Enter a name for this Content (if adding a Mad Mimi template, use the name of the template here)
  • Enter the html for your content. Customize the content by adding {lowercase_tags} - you will need to match the tag name in your SQL. Example of commonly used tags will be: {firstname} or {amountspent} -- remember lowercase for tags.
